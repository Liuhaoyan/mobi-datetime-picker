# mobi-datetime-picker
mobiscroll datetime custom theme

## Usage

```
    $(function () {
        $('#demo_datetime').mobiscroll().datetime({
            theme: 'ios',
            mode: 'scroller',
            display: 'top',
            lang: 'zh',
            minDate: new Date(2012, 3, 10, 9, 22),
            maxDate: new Date(2014, 7, 30, 15, 44),
            stepMinute: 5
        });
    });
```
